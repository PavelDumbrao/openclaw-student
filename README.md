# OpenClaw — Персональный AI-ассистент

Установка за 5 минут на Hostinger VPS.

---

## Что нужно

1. **Hostinger VPS** — тариф KVM 1 (~$4/мес), Ubuntu 22.04
2. **OpenRouter API ключ** — бесплатно на [openrouter.ai/keys](https://openrouter.ai/keys)

---

## Шаг 1 — Купи VPS на Hostinger

1. Зайди на [hostinger.com](https://hostinger.com) → VPS Hosting
2. Выбери **KVM 1** (самый дешёвый)
3. Операционная система: **Ubuntu 22.04**
4. После оплаты в панели управления найди:
   - **IP адрес** сервера (например: `123.45.67.89`)
   - **Пароль root** (или SSH ключ)

---

## Шаг 2 — Подключись к серверу

**На Windows** — скачай [PuTTY](https://putty.org) или используй PowerShell:
```
ssh root@123.45.67.89
```

**На Mac/Linux** — открой Терминал и введи:
```
ssh root@123.45.67.89
```

Замени `123.45.67.89` на свой IP адрес. Введи пароль когда спросит.

---

## Шаг 3 — Запусти установку

Скопируй и вставь эту команду в терминал (в окно SSH):

```bash
curl -sSL https://raw.githubusercontent.com/PavelDumbrao/openclaw-student/master/install.sh | bash
```

Нажми **Enter**.

Скрипт спросит OpenRouter API ключ — вставь его и нажми Enter.

Установка займёт 2-3 минуты. В конце ты увидишь ссылку вида:
```
http://123.45.67.89:18789/#token=abc123...
```

---

## Шаг 4 — Открой в браузере

Скопируй ссылку из терминала и открой в браузере на своём компьютере.

**Сохрани эту ссылку** — она нужна каждый раз для входа.

---

## Шаг 5 — Познакомься с ассистентом

При первом входе ассистент:
- Спросит как тебя зовут
- Расскажет что умеет
- Предложит подключить Brave Search (поиск в интернете)

---

## Получить OpenRouter API ключ

1. Зайди на [openrouter.ai/keys](https://openrouter.ai/keys)
2. Зарегистрируйся (можно через Google)
3. Нажми **Create Key**
4. Скопируй ключ (начинается с `sk-or-...`)

Бесплатный тариф включает доступ к модели Hunter Alpha.

---

## Проблемы?

**Не могу подключиться по SSH** — проверь IP адрес в панели Hostinger

**Скрипт не запускается** — убедись что ты подключён как `root`

**Ссылка не открывается** — проверь что порт 18789 открыт в файрволе Hostinger (VPS → Firewall → добавь порт 18789)
